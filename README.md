# SolNativa Energy - Landing Page de Conversión

Una landing page de alto rendimiento diseñada para **SolNativa Energy**, enfocada en la captación de leads (generación de prospectos) para soluciones de energía solar en la Riviera Maya.

## 🚀 El Proyecto
Este proyecto fue desarrollado con el objetivo de transformar una plataforma desactualizada en una herramienta de marketing moderna, rápida y orientada a la conversión, optimizada totalmente para dispositivos móviles.

## 🛠️ Stack Tecnológico
- **Framework:** [Astro](https://astro.build/) (Arquitectura de islas para máximo rendimiento).
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/) (Diseño *Mobile-First* y responsivo).
- **Interactividad:** [Alpine.js](https://alpinejs.dev/) (Para componentes ligeros como el menú hamburguesa).
- **Optimización:** Astro Assets (Optimización automática de imágenes).

## 🎯 Características Principales
- **Diseño Estratégico:** Estructura pensada para guiar al usuario hacia la conversión desde el primer *scroll*.
- **Mobile-First:** Interfaz totalmente adaptada a dispositivos móviles, garantizando una experiencia fluida en pantallas de cualquier tamaño.
- **CTA Directo:** Flujo de contacto optimizado mediante botones de acción inmediata (*Call to Action*) conectados directamente a WhatsApp.
- **Rendimiento Extremo:** Carga ultra rápida gracias a la arquitectura de Astro, eliminando *overhead* innecesario.
- **SEO & Accesibilidad:** Código semántico optimizado para buscadores y una estructura clara para lectores de pantalla.
